# 用户手册

当前版本可通过 `python main.py` 启动 GUI。

左侧导航包含仪表盘、流量监控、告警中心、规则管理、报告导出、系统设置。

## 导入 pcap

1. 打开“流量监控”页面。
2. 点击“导入 pcap”。
3. 选择本地 `.pcap`、`.pcapng` 或 `.cap` 文件。
4. 等待状态文字显示导入完成。

导入后，表格会显示时间、源 IP、目标 IP、协议、源端口、目标端口、长度和摘要。

## 实时抓包

1. 打开“流量监控”页面。
2. 点击“刷新网卡”。
3. 选择网卡，或使用“默认网卡”。
4. 点击“开始抓包”。
5. 需要停止时点击“停止抓包”。

Windows 下可能需要安装 Npcap 并以管理员权限运行。

## 规则管理

打开“规则管理”页面，可以启用或禁用规则，调整阈值和时间窗口，也可以编辑黑名单 IP。保存后，后续导入 pcap 或实时抓包会使用新的规则配置。

## 仪表盘

打开“仪表盘”页面，点击“刷新统计”可以查看当前数据库中的协议分布、告警等级分布、Top 源 IP 和 Top 目标端口。
