# Lightweight IDS 项目计划

## 项目目标

Lightweight IDS 是一个面向网络安全课程实践的轻量级入侵检测系统。项目目标是在桌面端完成流量导入、实时抓包、规则检测、告警管理、统计展示和报告导出，帮助学生理解 IDS 的基本数据流和检测逻辑。

## 已完成功能

1. 桌面 GUI：包含仪表盘、流量监控、告警中心、规则管理、报告导出和系统设置页面。
2. 数据采集：支持 pcap 文件导入和基于 Scapy 的实时抓包。
3. 协议解析：提取 IP、端口、协议、TCP flags、DNS 查询、HTTP 字段和 payload 摘要，并增强了 TLS 识别。
4. 数据存储：使用 SQLite 保存数据包、告警、内置规则、自定义规则和系统设置。
5. 检测规则：支持端口扫描、SYN Flood、ICMP Flood、敏感端口、黑名单、暴力破解、DNS 异常、HTTP 可疑请求、SQL 注入、XSS、恶意命令、异常外联、横向移动、主机扫描、TLS 指纹风险和轻量异常评分。
6. 自定义规则：支持按协议、IP、端口和关键词创建自定义规则。
7. 告警管理：支持筛选、搜索、详情查看、确认、忽略和 CSV 导出。
8. 统计分析：仪表盘展示协议分布、告警等级、Top IP、Top 端口、攻击链阶段、异常评分趋势和攻击链时间线。
9. 攻击链分析：按源 IP 关联扫描、利用、执行、C2、横向移动等阶段。
10. 误报降噪：提供白名单、资产重要性和重复告警合并的分析模块。

## 阶段状态

- 阶段一：基础骨架、数据模型、数据库、GUI 空页面，已完成。
- 阶段二：pcap 导入、Scapy 解析、PacketRecord 表格展示，已完成。
- 阶段三：检测引擎、规则基类和核心规则，已完成。
- 阶段四：告警中心、SQLite 保存、报告导出、`.gitignore`，已完成。
- 阶段五：规则管理、统计图表、实时抓包，已完成。
- 扩展阶段：特征匹配、异常行为、TLS 指纹、异常评分、攻击链分析、误报降噪和 GUI 分析视图，已完成。
