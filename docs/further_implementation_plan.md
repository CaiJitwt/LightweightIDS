# 后续实现完成情况

本文档记录 `further_implementation` 中各项扩展要求的完成状态。

## 阶段一：特征匹配检测

已完成：

- SQL 注入检测。
- XSS 检测。
- HTTP 可疑请求检测。
- 恶意命令检测。
- 暴力破解连接检测。
- DNS 异常检测。

## 阶段二：异常行为检测

已完成：

- 异常外联检测：识别内网主机访问公网非常规端口，以及固定间隔心跳行为。
- 横向移动检测：识别同一主机短时间访问多个内网目标的 SMB、RDP、SSH 等管理端口。
- 主机扫描检测：识别同一源 IP 短时间访问大量不同目标 IP。

## 阶段三：扩展能力

已完成：

- TLS 指纹风险：识别 TLS 摘要中的弱版本、弱加密套件和可疑证书关键词。
- 轻量异常评分：基于协议、端口、包大小和历史轮廓计算异常分数。
- 攻击链关联分析：按源 IP 串联扫描、利用、执行、C2、横向移动等阶段。
- 误报降噪：提供白名单过滤、资产重要性提权和重复告警合并模块。

## GUI 调整

已完成：

- 仪表盘新增攻击链阶段统计。
- 仪表盘新增异常评分趋势。
- 仪表盘新增攻击链时间线。
- 告警中心新增攻击链视图。
- 规则管理通过 SQLite 默认规则表自动展示新增规则。
- 主要新增视图的标题、列名和按钮文案已整理为稳定可读文本。

## 当前状态

项目已经具备课程项目要求中的完整主流程：

1. 导入或抓取流量。
2. 解析数据包。
3. 保存数据。
4. 执行内置规则和自定义规则。
5. 生成并管理告警。
6. 展示统计图表、攻击链和异常评分。
7. 导出报告。

后续如需继续增强，可优先考虑完整 JA3/JA4 指纹、基于真实数据集训练的异常检测模型，以及更细粒度的资产画像。
